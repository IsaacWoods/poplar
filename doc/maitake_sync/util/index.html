Reusable utilities for synchronization primitives.

This module contains utility code used in the implementation of the synchronization primitives provided by maitake-sync. To enable code reuse, some of these utilities are exposed as public APIs in this module, so that projects depending on maitake-sync can use them as well.

This module exposes the following APIs:

  • Backoff: exponential backoff for spin loops
  • CachePadded: pads and aligns a value to the size of a cache line
