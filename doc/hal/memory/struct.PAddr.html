Struct hal::memory::PAddr

source · 
#[repr(transparent)]
pub struct PAddr(_);
Expand description

Represents a physical address. If the target architecture has any requirements for valid physical addresses, they are always enforced.

source

pub const fn new(address: usize) -> Option<PAddr>

Construct a new PAddr. On x86_64, physical addresses must be less than 2^52; if this is not the case, this will return None.

source

pub fn align_down(self, align: usize) -> PAddr

Align this address to the given alignment, moving downwards if this is not already aligned. align must be 0 or a power-of-two.
